## Spring + SpringMvc + Mybatis集成简单个人博客系统

####  SSM框架，基于JavaConfig配置方式，不用XML配置文件
在使用Spring开发，经常会看到各种繁多的XML配置非常烦人，在Spring3之后，Spring支持使用JavaConfig来代替XML配置，这种方式越来越多人使用了。在SpringBoot中项目中，基本上见不到XML的影子了。具体的配置不多说，查看源代码。
